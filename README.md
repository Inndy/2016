#  STICON 2016 網頁

請多提供技術上及網頁設計上的意見

## 環境準備

npm 版本過舊可能會導致之後 build 失敗
[更新 node, npm 參考資料](https://nodejs.org/en/download/package-manager/)

安裝套件管理器至全域
```
npm install -g gulp
```

安裝 build 網頁時會用到的前處理器
```
npm install
```

## build

所有 source code 都在資料夾 *app* 下

build 網頁
```
gulp
```

debug
```
gulp debug
```
